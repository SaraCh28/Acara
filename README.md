# Acara 🎉

Acara is a Flutter-based event discovery application designed to help users find exciting events near them or anywhere in the world.

## Features

- User Signup / Login
- Secure Authentication
- Real Event Data via APIs
- Search Events by City / Country
- Location Detection
- Modern UI/UX
- Event Details Screen

## Tech Stack

- Flutter
- Dart
- Firebase / Supabase
- Ticketmaster API
- Geolocation Services

## Getting Started

This project is a starting point for a Flutter application.

### Prerequisites

- Flutter SDK
- Dart SDK
- A Firebase or Supabase project for authentication and data storage

### Running the app

```bash
flutter pub get
flutter run
```

For more resources see the Flutter documentation.
